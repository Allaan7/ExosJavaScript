Valeurs

Observez le programme valeurs.js et tentez de prévoir les valeurs finales de chaque variable.

var a = 2; la valeur de ma variable "a" est égale à 2

a = a - 1; la valeur de ma variable a est égale à "a" -1

a++; ajoute +1 à la valeur de ma variable a

var b = 8; la valeur de ma variable "b" est égale à 8

b += 2; la valeur de ma variable b est égale à sa valeur initiale +2

var c = a + b * b; la valeur de ma variable "c" est de 102

var d = a * b + b; la valeur de ma variable "d" est de 30

var e = a * (b + b) la valeur de ma variable "e" est de 40

var f = a * b / a; la valeur de ma variable "f" est de 10

var g = b / a * a; la valeur de ma variable "g" est de 10

Vérifiez vos prévisions en créant ce programme puis en le déboguant pour observer les valeurs des variables.
